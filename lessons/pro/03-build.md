**Concept:** First build-from-scratch for the pro track. The user picks one weekly task they currently do manually (status update, project summary, vendor email) and drives the agent to build a single-page tool that does it. By Lesson 3 they've fixed something, extended something, and now build something. The progression is intentional.

**Why build-from-scratch only at Lesson 3:** Lesson 1 (debug) and Lesson 2 (extend) teach the loop without the cognitive load of "what should I build." Lesson 3 introduces that load only after the loop is automatic.
